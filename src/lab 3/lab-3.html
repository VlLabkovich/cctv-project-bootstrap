Лабораторная работа №3

Распознавание автономеров



Цель работы:

Изучить принцип работы, настройки и применения детектора распознавания автономеров, получить необходимые навыки работы с видеоархивами,а также анализа полученных данных (сработок) детектора

Задачи:

Получить теоретические и практические навыки по настройке и контролю работы детектора распознавания автономеров, произвести сбор данных из зон детектирования после предварительной настройки детектора, на основе анализа полученных метаданных записанных в видеоархиве.


1.Теоретическая часть

Для успешного выполнения лабораторной работы требуется изучить и руководствоваться следующим теоретическим материалом:

  • Настройка архивов;
  • Настройка анализа видео из внешних систем (Offline Analytics)
  • Загрузка видео в ПК Axxon Next
  • Создание детекторов
  • Настройка детекторов
  • Настройка детекторов распознавания номеров
  • Детектор распознавания номеров
  • Настройка онлайн распознавания номеров
  • Создание списка номеров
  • Настройка автоматической реакции на распознавание номера из списка
  • Проверка срабатывания детектора
  • Видеонаблюдение в режиме анализа архива
  • Распознавание и поиск номеров транспортных средств
  • Поиск по номерам
  • Системный журнал


2.Практическое задание

После изучения теоретического материала и успешной сдачи теста контроля знаний, следует выполнить практическое задание для закрепления теоретических знаний и полуления практических навыков по работе с программным комплексом AxxonNext, настройке детектора распознавания автономеров, а также по работе с видеоархивам.

Задание: Произвести детектирование и распознование автономеров:

Видео 1: Произвести детекцию и распознавание 5 номерных знаков автомобилей

Видео 2: Произвести детекцию и распознавание номерных знаков 2-х автомобилей проезжающих под шлагбаумом

Произвести поиск и выборку метаданных работы детектора из видеоархива различными способами, для оценки корректной настройки детектора и полученных результатов поиска (см. выше. Видеонаблюдение в режиме анализа архива)

  1. Запустить программный комплекс AxxonNext(см.раздел Теория)
  2. Создать видеоархив требуемого размера
  3. Добавить в программный комплекс новое "программное устройство" для детектирования (в данном случае детектирование будет производиться посредством оффлайн-aналитики с помощью встроенной функции в ПО AxxonNext "Анализ видео из внешних систем"). Видеофайлы, которые следует загрузить в программный комплекс AxxonNext для выполнения лабораторной работы находитяся в папке: ЭУМК-2020/Видео/3. Распознавание автономеров/Video_1 и Video_2. (см. выше. Загрузка видео в ПК Axxon Next)
  4. Сделать привязку добавленного "программного устройства" к созданному ранее видеоархиву
  5. Создать базовый обьект, детектор лиц.
  6. Произвести настройку детектора лиц заданием определенных характеристик. Выделить детектируемую область в окне видеонаблюдени, также можно выделить область которую нужно исключить из детекции
  7. Настроить макрокоманду (автоматическое правило) при срабатывании детектора, режим инициирования тревоги и записи в архив (в параметре завершить после, желательно указывать до 5 сек) (см.раздел Теория)
  8. С помощью системного журнала проверить правильность работы детектора и макрокоманд, проведя процедуру поиска событий. (см. выше. Загрузка видео в ПК Axxon Next
  9. Выполнить основное задание, после настройки программного комплекса

После выполнения всех вышеизложенных пунктов, с полученными данными обратиться к преподавателю, с целью оценки результатов выполнения лабораторной работы.
